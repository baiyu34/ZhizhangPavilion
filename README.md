# 知章阁（一个简单的文章管理系统）

#  包含功能：用户注册与登录，添加、修改和编辑文章分类，增加、修改及删除文章，文章搜索功能，用户个人信息及密码更改等

#  关于项目：
环境：jdk17.0.10 + mysql8.0.36 + maven  + IDEA
后端：springboot3 + springmvc + mybatis
前端：Vue3 + element-plus
